# Multiclone

Multiclone is a Discord bot that allows for copying and pasting many Discord features across channels and servers.

## Clonable Features

### Servers
 - Bans
 - Channels (text, voice, category)
 - Channel permissions (only when copying everything)
 - Emojis and Stickers
 - Roles

### Channels
 - Permissions
 - Threads
